Generation parameters of various other properties

The following properties are piecewise uniformly generated. For each age there is the rate of Uniform Distribution picked up by the polynomial function.
For example, configure the constant 0 function for Emergency Rate would return only non Emergency patients, e.g. date of registration is at least one day before admission.
